Sistema Spirit Of Fire.



Permite la creación de:



* Habitaciones
* Huéspedes
* Reservas 
* Productos (Minibar)
* Categorías



Permite a su vez editar y eliminar los objetos en estos módulos. El programa fue desarrollado con el framework Django 5.2 y MySQL, siguiendo el patrón MVC (Modelo-Vista-Controlador)
